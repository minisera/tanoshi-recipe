# 開発環境
* Ruby 2.6.6
* Rails 6.0.0
* macOS Catalina 10.15.7

# 起動するにあたり
お忙しい中恐縮ですが、起動するにあたり、以下の手順にてお進みください。

1. 解凍後、`tanoshi-recipe`ディレクトリに移動します。
2. `bundle install`をします。
3. `rails db:create`をします。
4. `rails s`をします。
5. Google Chromeにて`http://localhost:3000`にアクセスします。

それでは、よろしくお願い致します。
